func genProofIdx ¶ 

func genProofIdx(root *TreeNode, leafIdx int) ([][]int, error)

genProofIdx generates proof indices for the leaf node corresponding to the given leaf index. It traverses the Merkle tree from the root to the leaf node, collecting the left and right indices of each node in the proof path and appends them to the result.
